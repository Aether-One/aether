# Aether — Development

## Requirements

- Node.js 20+
- npm

## Setup

```bash
git clone https://github.com/aether-one/aether
cd aether
npm install
```

## Running

```bash
npm run dev
```

## Building

```bash
npm run build
```

## Flags

| Flag | Description |
|------|-------------|
| `--no-animation` | Skip startup animation |

## Dependencies

| Package | Purpose |
|---------|---------|
| chalk | Terminal colors (hex support) |
| tsx | TypeScript execution (dev) |
| typescript | Compiler |
